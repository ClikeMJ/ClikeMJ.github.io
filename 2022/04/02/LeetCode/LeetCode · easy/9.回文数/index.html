9.回文数

给你一个整数 x ，如果 x 是一个回文整数，返回 true ；否则，返回 false

回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数

  • 例如，121 是回文，而 123 不是

方法一：数字转字符串比较

  • 整数转为字符串，循环一半长度判断对应元素是否相等即可
  • 或翻转整个字符串对比

方法二：数字本身反转

  • 注意：如果反转后的数字大于int所能表示的最大范围，将遇到整数溢出错误
  • 可以用long类型防止溢出，long revers = 0;

方法三：反转一半数字

  • 通过除10和取余来获取反转一半的数字，避免整数溢出
  • 分偶数位和奇数位分别比较前后一半数字
  • 时间复杂度O(log10(n))、空间复杂度O(1)

方法四：获取数量级

  • 首先获取当前数量级n=10^⌊log⁡10x⌋^。eg：x = 24， n = 10；x = 6234， n = 1000
  • 通过x / n 获取首位，通过x % 10 获取末位
  • (x % n) / 10，实现去除首位和末位，（x % n去除首位，x / 10去除末位）
  • n /= 100 ，实现x位数减2

总结：

  • int类型可能出现整数溢出的情况，可以用long类型解决
  • 类似回文比较时，用前指针和后指针比较left<right，可以减少一半循环
  • 折半处理时，要考虑单独处理个数为奇数的情况
  • 注意：Math函数中，返回值为double类型的情况，可能需要强制类型转换
